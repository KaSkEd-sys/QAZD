## Quick Start
```bash
python main.py your_program.qazd
```

## Installation
```bash
git clone https://github.com/KaSkEd-sys/QAZD
cd QAZD
pip install -r requirements.txt
```

Full documentation is available [here](docs/docs.md)

## Memory Cells

| Cell | Purpose |
|------|---------|
| A, B, C, D, E, F | General purpose |
| Z | System (interpreter control) |
